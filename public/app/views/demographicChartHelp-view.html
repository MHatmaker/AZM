Charts Help

The Charts section presents an alternate perspective of the summary data. The charts integrate data from both the Census 2010 and American Community Survey (ACS) 5 year estimates for 2007 - 2011. Chart titles ending with an '*' have Census 2010 as a data source while titles ending in '^' have the ACS estimates as their data source.

Clicking the Display Chart Legend box will bring up the chart legend for the data being displayed.

Hovering over the chart will bring up a tool tip showing the data value and percentage for each section of the chart.

The Charts can be exported to an PDF data file using the Export Report to PDF button. This export button will export a report that includes all of the listed charts and the summary report to a PDF document.
