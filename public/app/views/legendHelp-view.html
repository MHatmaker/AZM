Legend Help

The legend describes the contents of the current map. The content of the map is organized into layers that group similar types of geographic features.

The map contains the following layers:
  • The current demographic map. This is the default layer. The description of the current map is displayed at the top followed by the classes contained within the map. Each class describes the shade on the map used to display a range of values.

    • The rest of the layers are located in the drop-down panel labeled Layer Options.

  • Census tract labels
  • Street map information
  • Aerial photo
  • Basemap information such as city boundaries and terrain.

You can change the display of the map's content in the following ways:

  • Change the visibility of a layer by checking or un-checking the layer's check-box.
  • Change the transparency of the census block group layer by moving the transparency slider. Moving the slider to the left will increase the layer's transparency which will allow for seeing the contents underneath the layer.
  • To change the current demographic map, choose an item from the maps menu.
