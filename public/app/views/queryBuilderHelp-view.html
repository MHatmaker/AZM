Advanced Query Help

The Advanced Query builder will allow you to select features by using individual feature criteria from the subject data listed in the drop-down menu. When you choose the feature attributes you want to use in the query, you can then specify a range of values to use to define the selection set. You can choose a single field or choose multiple fields and groups.

  1. Use the drop-down menu selector to select a subject to query. The data type selected in the subject changes the values that can be selected for operation.
  2. Use the drop-down menu selector to select the Operation type.
  3. Enter desired value in into value text box.
  4. Additional search parameters or conditions can be added by using the additional conditions drop-down selector menu. When you specify multiple conditions, it's important to understand the difference between the And and Or operators.
    1. The And operator is restrictive. It only retrieves data that mets all the conditions you specify. See #1 below.
    2. The Or operator is more flexible. It returns data that meets any of the conditions you specify. See #2 below.

  5. To remove a search parameter or condition just click the red X at the end of the condition row.
  6. Use the Verify Query button to test your query. It will return the number of block groups identified by query statement.
  7. Use the Run Query button to run the query once you are satisfied with query statement.
  8. The Cancel button closes out of the Advanced Query window.

The source for the reports is Census 2010 and American Community Survey 2008-2012.
