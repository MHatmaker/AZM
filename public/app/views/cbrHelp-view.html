Maps Help

Choose a Map

A menu of available thematic demographic maps is provided in the Maps panel. The maps are organized by topic (e.g., Population, Housing) and sub-categories (e.g., Age, Population by Race). Note that changing a map will change the text in the viewer's header and the contents of the Legend panel.

Advanced Map Options

Data classes - This provides the ability to specify the number of intervals the data is divided up into. Choosing the number of data classes is an important part of map design. Increasing the number of data classes will result in a more "information rich" map by decreasing the amount of data generalization. However, too many data classes may be overwhelming, providing to much information and distracting the map reader from seeing general trends in the distribution. In addition, a large numbers of classes may compromise map legibility as more classes require more colors that become increasingly difficult to tell apart.

Classification Type - A means of generalizing the maps by sorting or arranging data into groups or categories on a map. You can explore the data by applying the different classification techniques. The classification method you choose depends on the nature of your data, and what you want to show about the data. When you classify your data, you can either use one of the standard classifications schemes provided or create custom classes based on class ranges you specify.

  • Natural Breaks - A method of manual data classification that seeks to divide data into classes based on natural groups in the data distribution. Natural breaks occur in the histogram at the low points of valleys. Breaks are assigned in the order of the size of the valleys, with the largest valley being assigned the first natural break.
  • Equal Interval - A data classification method that divides a set of attribute values into groups that contain an equal range of values.
  • Quantile - A data classification method that distributes a set of values into groups that contain an equal number of values.
  • Custom - A method that lets the user choose the appropriate or meaningful classification values.

Modify Button - This button brings up a menu that allows the user to modify classification values.

Color Scheme - A range of colors used to show ranking or order among classes on a map.

For more information on color schemes and data classification visit - www.colorbrewer2.org
