Selected Features Help

The Selected Features presents an aggregate view of the selected block groups: there is one record for each summary metric. The results in this table integrate data from both the Census 2010 and American Community Survey (ACS) 5 year estimates for 2007 - 2011. Records ending with an '*' have Census 2010 as a data source while records ending in '^' have the ACS estimates as their data source.

The number of selected features returned by Custom Summary or Advanced Query is displayed at the top of the table.

Selected features can also be sorted by each column in ascending and descending order by clicking on each column header.

Hovering over a row will highlight the Census Block Group it represents on the map.

By clicking on a row you can highlight the individual row.

The selected features can be exported to an Excel or CVS data file using the Export Results selector and button.
