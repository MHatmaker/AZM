Markup Tools Help

The Markup Tools give a user drawing and markup capabilities with settings for different shapes and colors.

The following markup shapes are available:

  • Polygon - Click on the map to start drawing a side, single click to change drawing direction, double-click to finish drawing.
  • Circle - Click to add a simple circle, or click, hold, and drag the cursor to create a custom size circle.
  • Arrow - Click to add a simple arrow pointing East, or click, hold, and drag the cursor to create a custom size arrow pointing in any direction.
  • Freehand Polygon - Click, hold, and drag the cursor to create a custom shape. Release click to finish drawing.
  • Text Box - Enter text into Text Input box, select color and fontsize and then click on the map to add textbox.

The following settings can be set for each shape. Be sure to change these settings before drawing the shape:

  • Fill Color - Change the color used to fill shape. For Text, the fill shape determines the text color.
  • Transparency Slider - Changing the transparency slider affects the transparency of the fill color.
  • Outline Color - Change the color used as the outline of shape. Does not apply to Text.

The following tools are available to edit existing markups:

  • - Edit Markup - Allows you to select a markup on the map. Selections can be resized and rotated.
  • - Delete Markup - Deletes the selected markup from the map. Click this icon first, then select the markup(s) to be deleted.
  • - Undo Last Markup - Removes the last drawn markup from map. Continue clicking to remove markups in the reverse order in which they were drawn.
